# Contacttura

## Sobre

### O projeto Contacttura consiste em desenvolver uma Api RestFull, que será realizar um CRUD de uma agenda de contato.

## Features

- [] Criar novo contato
- [] Listar todos os contatos
- [] Buscar por Id
- [] Atualizar dados de um contato
- [] Remover um contato pelo Id
